# CSI Intrusion Detection using ESP32

Real-time WiFi CSI intrusion detection using:

- ESP32-E passive CSI sensing
- CNN + LSTM deep learning
- TensorFlow GPU
- Real-time dashboard visualization
- CSI amplitude + phase feature extraction

## Features

- Passive WiFi sensing
- Real-time intrusion probability
- Live CSI heatmaps
- CNN+LSTM inference
- ESP32 CSI streaming

## Environment

- Ubuntu 26.04 LTS
- Python 3.10
- TensorFlow GPU
- ESP-IDF v4.4.4
